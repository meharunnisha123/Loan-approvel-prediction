# Loan-approvel-prediction

The Loan Approval Prediction Project aims to analyze historical loan application data and predict whether a loan application will be approved or rejected. The project uses data analytics and machine learning techniques to identify key factors influencing loan approval decisions, such as income, loan amount, CIBIL score, education, employment status, and asset values.

The project was developed using Excel, SQL, Python, Power BI, and Tableau to perform data cleaning, analysis, visualization, and predictive modeling.
